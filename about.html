About

Welcome to Team Charlie's group project for the EECS 1012 Winter Term! Our project revolves around a movie rating website. Users can rate any of the site's 18 movies using a 5-star rating system. Behind the scenes, our server uses key-value pairs to store the data and calculate the average rating for each movie, when the user selected a star and clicks on the submit button. This will then be displayed on a movie page. This project was created using node.js. To get started, simply execute the command "node server.js" to launch the server on port 8080. Please note that user ratings will be cleared from the server upon restart, ensuring a fresh start for each session."
